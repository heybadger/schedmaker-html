# 📅 Class Sched

A clean and beginner-friendly web application for creating, managing, and exporting your weekly class schedule.

Designed with a minimal interface and responsive layout, **Class Sched** lets students quickly build their timetable, save it, import it later, or export it as an image for sharing.

---

## ✨ Features

- 📝 Create a custom class schedule
- 📅 Support for multiple days per subject
- ⏰ Custom start and end times
- 📋 Personalized schedule title
- ❌ Remove individual courses
- 🗑️ Clear entire schedule
- 📤 Export schedule as JSON
- 📥 Import previously saved schedules
- 🖼️ Save schedule as PNG image
- 🎨 Add decorative emojis before exporting
- 🖨️ Print-ready schedule
- 📱 Fully responsive (Desktop, Tablet & Mobile)
- 💡 Beginner-friendly interface with quick start guide

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/heybadger/class_sched.git
```

### Open the project

Simply open `index.html` in your preferred web browser.

No installation or build tools required.

---

## 🎯 How to Use

1. Enter a schedule title.
2. Input your course name.
3. Select one or more class days.
4. Choose the start and end time.
5. Click **Add Course**.
6. Continue adding subjects until your schedule is complete.
7. Export your schedule as:
   - JSON (backup)
   - PNG Image
   - Print
